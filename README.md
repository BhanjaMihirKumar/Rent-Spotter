# Rent-Spotter

Rent Spotter is a full-stack web application that helps users search and explore rental properties based on city, area. It provides a responsive interface with property listings and advanced search functionality for a seamless house-hunting experience.

## 🚀 Features

- User Authentication
- Search Properties by City, Area 
- Property Listings
- Responsive Design
- CRUD Operations

## 🛠 Tech Stack

**Frontend:** React.js, HTML, CSS, JavaScript  
**Backend:** Node.js, Express.js  
**Database:** MySQL  
**Tools:** Git, GitHub, Postman

## 📂 Installation

```bash
git clone https://github.com/your-username/Rent-Spotter.git

cd Rent-Spotter

npm install

npm start
```
